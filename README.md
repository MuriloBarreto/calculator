# 📟 Calculadora

Projeto de uma calculadora feita em react 

### 🤔 Como Usar

```
   - Clone esse repositório:
   $ git clone https://github.com/MuriloBarreto/calculator.git

   - Entre no diretório:
   $ cd calculator

   - Instale as dependências:
   $ npm install

   - Inicie o projeto: 
   $ npm start
  ```
